# Use Cases

## Use Case 1: Recurring Contractor Billing Packet
A finance operator runs the monthly cycle for a contractor roster and generates packet-ready artifacts with consistent structure.

## Use Case 2: Mixed Billing Model Operations
Teams manage contractors with different service profiles while preserving uniform output quality and review-ready totals.

## Use Case 3: Fast Rerun After Input Correction
If source context changes, the operator reruns the flow quickly and receives updated artifacts without manual reconstruction.

## Use Case 4: Audit-Oriented Document Operations
Reviewers move from source row to output folder directly, reducing evidence search time during internal or external checks.
