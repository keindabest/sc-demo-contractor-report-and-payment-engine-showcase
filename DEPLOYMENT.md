## Quick Try Path
1. Open the deployed application.
2. Select contractor, customer, month, and year in the Generation tab.
3. Click Jira sync and wait for successful validation state.
4. Click `Create Documents`.
5. Open the linked Drive folder and inspect generated packet files.

## Expected Result
You get a complete payout packet in the target output folder with direct traceability from selected input context to generated artifacts.

## Boundary Notes
This try path demonstrates workflow behavior in a public-safe context. Screenshots and sample files are synthetic; secrets and private production internals are excluded.
