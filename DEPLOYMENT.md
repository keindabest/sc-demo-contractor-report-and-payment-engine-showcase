## Quick Try Path
1. Open the deployed application.
2. Select contractor, customer, month, and year.
3. Run the sync step and confirm context is loaded.
4. Trigger generation.
5. Open the Drive folder and inspect outputs.

## Expected Result
A complete billing packet appears in the target output folder with traceable linkage from input context to generated artifacts.

## Boundary Notes
This try path demonstrates workflow behavior in a public-safe context. Internal-only configuration and production-sensitive details are intentionally excluded.
